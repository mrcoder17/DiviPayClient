DiviPay - это десктопное приложение, разработанное Боксбергером Артемом, студентом 2 курса ФИТ НГУ. Оно представляет собой удобное решение для осуществления совместных покупок и управления финансами в сообществе. Архитектура приложения основана на клиент-серверной модели, где серверная часть реализована с использованием Spring Boot, а клиентская часть разработана на JavaFX. Взаимодействие между клиентом и сервером осуществляется с помощью RESTful API.

Основные функциональные возможности приложения включают:

Авторизация и регистрация: При запуске приложения, пользователю предлагается окно авторизации, где он может ввести свои учетные данные для входа в систему, либо зарегистрироваться, указав необходимую информацию.

Мой профиль: После успешной авторизации, пользователь может просмотреть и редактировать свой профиль. Он может загрузить аватарку, добавить дополнительную информацию о себе и настроить данные профиля.

Контакты: Пользователь имеет доступ к списку контактов, где отображаются другие участники сообщества.

Запросы: Пользователь может создавать запросы на покупку определенных товаров. Запросы отображаются в списке и могут быть просмотрены и обработаны другими участниками сообщества.

Покупки: Пользователь может посмотреть весь список покупок и внести свою.

Главная страница: На главной странице отображается общая информация о сообществе, список покупок и актуальных запросов.

DiviPay предлагает простой и интуитивно понятный интерфейс, который позволяет участникам сообщества управлять своими покупками, делиться затратами и взаимодействовать друг с другом. Это позволяет оптимизировать процесс совместных покупок и создать удобное средство управления финансами в рамках сообщества.
